# 🦅 ScEagle

**ScEagle** — расширение для браузера, которое определяет фишинговые сайты по возрасту домена и выявляет подделки известных сервисов.
#Возможности

- Проверяет возраст домена через WHOIS
- Показывает предупреждение для молодых доменов (< 30 дней)
- Красный баннер + звук для критически опасных (< 3 дней)
  > *P.s:* в новых версиях браузеров звук срабатывает только после взаимодействия со страницей. Будет исправлено в следующих обновлениях.
- Проверяет на подделку известных сайтов (тайпсквоттинг)

#Установка

1. Скачайте или склонируйте репозиторий
2. Откройте `chrome://extensions/`
3. Включите **"Режим разработчика"**
4. Нажмите **"Загрузить распакованное расширение"**
5. Выберите папку с расширением

#Как это работает

Расширение обращается к серверу `https://sceagle.minbed.ru`, который выполняет WHOIS-запрос и возвращает дату регистрации домена.  
*Вы можете указать свой домен.*

#Что используется

- JavaScript, HTML, CSS
- Python (WHOIS-сервер)
- Nginx (прокси + SSL)

#Развертывание сервера

1. Загрузите расширение в браузер
2. Запустите Python-сервер
3. В nginx (`/etc/nginx/sites-available/`) добавьте конфиг (рабочий вариант прилагается)
4. Установите сертификаты:
   `apt install certbot python3-certbot-nginx -y`
   получение сертификатов:
   `certbot --nginx -d sceagle.minbed.ru`
5. В конфиге укажите пути к полученным сертификатам
6. Откройте порты 80 и 443 в настройках роутера/фаервола
7. Перезагрузите страницу — всё готово!
